# iChat-App

Simple Chat App using custom APIs backed by Retrofit.


### Features
- List of User for chat
- Complete list of old messages
- User can send messsages to any of the users from list

### Topic covered
- Java (Primary Language)
- Retrofit
- Recycler View
- MVVM
- Live data
